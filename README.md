# media-bot

A Python project built with Poetry and Python 3.11.

## Installation

```bash
poetry install
```

## Usage

Run the application as a module:

```bash
poetry run python -m app
```

Or activate the virtual environment and run directly:

```bash
poetry shell
python -m app
```

## Project Structure

```

├── app/
```
